# ConsoleApp30
Завдання 1:
Розробіть додаток для роботи з дробами. Дріб може
відображатися у вигляді звичайного дробу (чисельник і
знаменник), десяткового дробу. Додаток має дозволяти
виконувати операції з дробами:
 додавання;
 віднімання;
 множення;
 ділення;
 скорочення дробів.
Використовуйте MVC-патерн для вирішення завдання.
Завдання 2:
Додайте до першого завдання можливість завантаження даних
із файлу. У файлі міститься інформація про дріб. Зверніть увагу,
що файли можуть бути з різним форматуванням. В одному
випадку дані можуть розділятися через лінію із символів "+". В
іншому випадку — через лінію із символів "^" і т. д.
Враховуйте цей факт при проєктуванні рішення.
Завдання 3:
Створіть клас «Car», в якому зберігатиметься така інформація:
 назва авто;
 колір;
 виробник;
 рік випуску;
 об'єм двигуна.
Реалізуйте необхідні методи і поля. Реалізуйте виведення
інформації на екран та у файл. Використовуйте MVC-патерн у коді
цієї програми.
Завдання 4:
Додайте до третього завдання можливість завантаження даних
із файлу. У файлі міститься інформація про велику кількість «Car».
Зверніть увагу, що файли можуть бути з різним форматуванням.
В одному випадку об'єкти «Car» можуть розділятися порожнім
рядком, а в іншому — через два порожні рядки і т. д.
Враховуйте цей факт при проєктуванні рішення.
